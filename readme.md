### Goal :dart:

-   Deliver mail to correct mailbox
-   Avoid falling objects

### Controls :joystick:

|  Key   |   Action   |
| :----: | :--------: |
| Arrows |    Move    |
|   O    | Throw Mail |
|   X    | Swap Color |
