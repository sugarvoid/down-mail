### Down Mail (work in progress)

![down mail cart](https://github.com/sugarvoid/down-mail/blob/master/down_mail%5Bv0.1%5D.p8.png)

### Goal :dart:

-   Deliver mail to correct mailbox
-   Avoid falling objects

### Controls :joystick:

|  Key   |   Action   |
| :----: | :--------: |
| Arrows |    Move    |
|   O    | Throw Mail |
|   X    | Swap Color |
