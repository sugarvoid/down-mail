## Goal :dart:

-   Deliver letters to correct customers
-   Avoid falling objects
-   Crashing into mailboxes is bad
-   Dogs will slow you down
-   See a demon, kill it with a letter quickly

## Controls :joystick:

|  Key   |   Action   |
| :----: | :--------: |
| Arrows |    Move    |
|   O    | Throw Letter |
